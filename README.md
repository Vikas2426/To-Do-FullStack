# Full Stack To-Do App

- **Mongo**
- **Express**
- **React**
- **Node**

---

## Installation

1. Clone this repository

2. cd to-do-express directory

   1. `npm install`
   1. `node server.js`

**If the server throws an error just rename the test.env --> .env**

3. cd to-do-react

   1. `npm install`
   1. `npm start`
